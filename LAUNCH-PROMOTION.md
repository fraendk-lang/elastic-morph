# Elastic Morph — Promotion (Launch)

Stand: Phase 1 · Demo: **Trip-Hop-Track (~90 s)**

---

## Demo-Track: Schnitt-Empfehlung

**Ja — auf ca. 75–90 Sekunden schneiden.** Nicht kürzer als 45 s (Analyse braucht etwas Material), nicht länger als 90 s (Download + erster Eindruck).

### Ideale Struktur (90 s)

| Sek | Inhalt |
|-----|--------|
| 0–8 | Direkt **im Groove** starten (kein langes Fade-in) |
| 8–45 | Verse / Aufbau — Visual atmet mit |
| 45–70 | **Hook, Break oder Drop** — stärkster Wow-Moment |
| 70–90 | 1–2 Bars ausklingen oder Loop-Punkt (für Canvas 8 s Loop später) |

### Technisch

- Format: **MP3**, 192–256 kbps, Stereo
- Dateiname: `assets/demo/elastic-morph-demo.mp3`
- `demo.json`: echten Titel + Künstler eintragen
- Prüfen: `npm run check:demo`

### Warum Trip-Hop gut passt

- Passt zu **Liquid Memory**, **Black Bloom**, Vinyl-Look
- Langsamer BPM = cinematisch, nicht hektisch
- Wichtig: **Snare/Kick/Hats** sollten hörbar sein — sonst reagiert das Visual zu schwach
